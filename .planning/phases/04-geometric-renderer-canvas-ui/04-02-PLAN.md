    'use client' component. Props:

    - `animated: boolean` — true for progressive build, false for instant
    - `onRenderComplete?: () => void` — callback when rendering done

    Implementation:
    - useRef<HTMLCanvasElement> for canvas reference
    - useEffect depending on [scene, animated]:
      - Get 2D context
      - HiDPI scaling: `const dpr = window.devicePixelRatio || 1; canvas.width = scene.width * dpr; canvas.height = scene.height * dpr; ctx.scale(dpr, dpr);`
      - If !animated: call drawSceneComplete(ctx, scene), fire onRenderComplete, return
      - If animated: implement progressive draw loop:
        - totalDuration = 750ms (within 0.5-1s range per CONTEXT.md)
        - fadeInDuration = 100ms per element
        - staggerDelay = totalDuration / scene.elements.length
        - requestAnimationFrame loop: for each element, compute start time and alpha progress
        - Each frame: clear + redraw background, then draw all visible elements with current alpha
        - On final frame: drawSceneComplete for clean final render, fire onRenderComplete
        - Return cleanup: cancelAnimationFrame(animationId) + aborted flag

    Canvas element:
    - style={{ width: scene.width, height: scene.height }} for CSS sizing
    - className: "rounded-lg max-w-full" + passed className
    - CSS: w-full on mobile via Tailwind, max-w-lg on desktop
    - aria-label="Generated geometric artwork"
