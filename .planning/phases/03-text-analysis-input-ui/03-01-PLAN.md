  <implementation>
    Phase 1 — Install dependencies:
    ```bash
    npm install afinn-165 syllable
    ```

    Phase 2 — Create sentiment module (src/lib/analysis/sentiment.ts):
    - Import afinn165 from 'afinn-165'
    - Define NEGATORS set: 'not', 'no', 'never', 'neither', "n't", "don't", "doesn't", "didn't", "won't", "wouldn't", "couldn't", "shouldn't", "isn't", "aren't", "wasn't", "weren't", "haven't", "hasn't"
    - computeSentiment(words: string[]): { polarity: number; magnitude: number }
    - For each word: lowercase, strip non-alpha except apostrophe/hyphen, check Object.hasOwn(afinn165, word)
    - 1-word lookback negation: if previous word is in NEGATORS, flip score sign
    - polarity = totalScore / wordCount (comparative)
    - magnitude = absMagnitudeSum / sentimentWordCount (average absolute per sentiment word, 0 if none)

    Phase 3 — Create syllable module (src/lib/analysis/syllables.ts):
    - Import { syllable } from 'syllable'
    - computeSyllableFeatures(words: string[]): { syllableVariance: number; syllableComplexity: number }
    - counts = words.map(w => syllable(w))
    - syllableComplexity = mean of counts
    - syllableVariance = population variance of counts

    Phase 4 — Create main analyzer (src/lib/analysis/text.ts):
    - analyzeText(text: string): Record<string, number>
    - Handle empty text: return all-zeros object
    - Split text into chars, words (split on whitespace, filter empty), sentences (split on [.!?]+)
    - Compute all 31 signals. Re-use logic from extractMockSignals for the statistical signals (charCount, wordCount, avgWordLength, etc.) — these were correct in the mock.
    - For sentiment: call computeSentiment(words)
    - For syllables: call computeSyllableFeatures(words)
    - For consonantClusterFreq: count matches of /[bcdfghjklmnpqrstvwxyz]{3,}/g in lowercased text, divide by (letterCount / 10)
    - For imperativeRatio: curated list of ~50 common imperative verbs (go, run, make, try, add, remove, click, open, set, get, put, take, find, use, keep, let, stop, start, move, turn, read, write, check, close, send, show, tell, give, ask, call, play, work, help, fix, cut, draw, push, pull, pick, drop, fill, clear, save, load, mark, list, sort, join, split, build). Check if sentence starts with one of these (case-insensitive). imperativeRatio = imperativeSentences / sentenceCount.
    - For clauseDepth: count commas + semicolons + occurrences of "which"/"that"/"who" as separate words, divide by sentenceCount
    - vocabRichness: same log(wordCount) scaling as mock
    - vocabularyRichness: alias of vocabRichness (both keys present)
    - paragraphBalance: same logic as mock (1 / (1 + variance(paragraphLengths) * 0.01))

    Phase 5 — Create barrel export (src/lib/analysis/index.ts)

    Phase 6 — Update calibration.ts:
